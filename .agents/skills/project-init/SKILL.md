# Project Initialization

## Purpose

PLAN.md をもとに
業務開始に必要なドキュメントを作成する。

## Procedure

1. PLAN.md を確認する

2. PLAN が存在しなければ
   人間へ作成を依頼する

3. SPEC.md の初版を作成する

4. 不足情報を質問する

5. SPECが完成するまで繰り返す

6. TODO.md を作成する

7. タスクを実行可能な粒度まで分割する

8. 各タスクへ担当者を付与する

- [AI]
- [HUMAN]

9. KNOWLEDGE.md を初期化する

## Rules

- PLANが無い状態でSPECを作成しない
- SPECが固まるまでTODOを確定しない

## Completion

以下が存在すること

- PLAN.md
- SPEC.md
- TODO.md
- KNOWLEDGE.md
